## Quick Answers
- **O que é redação segura de documentos?** Um processo que remove ou mascara permanentemente dados confidenciais de um documento, de modo que não possam ser recuperados.  
- **Quais plataformas são suportadas?** Tanto .NET quanto Java, cobrindo mais de 30 formatos de arquivo.  
- **Preciso de software externo?** Não — o GroupDocs.Redaction funciona sem Microsoft Office, Adobe Acrobat ou outras ferramentas de terceiros.  
- **Posso redigir PDFs e imagens?** Sim, você pode redigir texto, metadados, imagens, anotações e até páginas inteiras.  
- **É necessária licença para produção?** Uma licença válida do GroupDocs.Redaction é necessária para uso em produção; uma avaliação gratuita está disponível.

## What is Secure Document Redaction?
Redação segura de documentos é a técnica de excluir ou obscurecer permanentemente informações sensíveis — como identificadores pessoais, dados financeiros ou comentários confidenciais — de arquivos digitais. Ao contrário de simples ocultação, a redação garante que os dados não possam ser recuperados por nenhum meio, atendendo a padrões de conformidade como GDPR, HIPAA e PCI‑DSS.

GroupDocs.Redaction fornece uma API unificada para redação de documentos em várias plataformas. Aqui estão algumas razões convincentes para escolher nossa solução:

### Opções Abrangentes de Redação
- Redigir texto usando frases exatas, expressões regulares ou correspondência sensível a maiúsculas/minúsculas  
- Limpar propriedades de metadados, comentários e informações ocultas  
- Sanitizar ou remover completamente imagens com conteúdo sensível  
- Redigir anotações e comentários que possam conter informações confidenciais  
- Remover páginas inteiras ou intervalos de páginas contendo material sensível  
- Aplicar políticas de redação personalizadas para tipos específicos de documentos  

### Foco em Privacidade e Segurança
- Remoção permanente de informações sensíveis sem possibilidade de recuperação  
- Rasterização opcional para converter documentos em PDFs baseados em imagem  
- Recursos de proteção contra adulteração para impedir modificações não autorizadas  
- Sanitização completa do documento, incluindo metadados e propriedades ocultas  

### Sem Dependências Externas
GroupDocs.Redaction funciona sem exigir instalações de softwares externos como Microsoft Office, Adobe Acrobat ou outras ferramentas de terceiros.

## Frequently Asked Questions

**Q: O GroupDocs.Redaction suporta arquivos protegidos por senha?**  
A: Sim. Você pode abrir PDFs, Word ou arquivos Excel criptografados fornecendo a senha durante a etapa de carregamento.

**Q: Posso redigir conteúdo em massa em vários documentos?**  
A: Absolutamente. A API permite percorrer uma coleção de arquivos e aplicar a mesma política de redação a cada um.

**Q: A rasterização é opcional?**  
A: Sim. Você pode escolher rasterizar todo o documento ou apenas páginas específicas quando precisar de uma saída somente em imagem para maior segurança.

**Q: Quais linguagens de programação são suportadas?**  
A: Os SDKs principais estão disponíveis para .NET (C# / VB.NET) e Java. Bibliotecas wrapper existem para outras plataformas também.

**Q: Como garantir a conformidade da redação com o GDPR?**  
A: Use os recursos integrados de remoção de metadados e exclusão permanente de conteúdo; a API garante que os dados redigidos não possam ser recuperados, atendendo ao requisito do GDPR de “direito ao esquecimento”.
