## Quick Answers
- **¿Qué es la redacción segura de documentos?** Un proceso que elimina o enmascara permanentemente los datos confidenciales de un documento para que no puedan recuperarse.

## ¿Qué es la Redacción Segura de Documentos?
La redacción segura de documentos es la técnica de eliminar o oscurecer permanentemente información sensible —como identificadores personales, datos financieros o comentariosenciales— de archivos digitales. A diferencia de una simple ocultación, la redacción garantiza que los datos no puedan recuperarse por ningún medio, cumpliendo con normas de cumplimiento como GDPR, HIPAA y PCI‑DSS.

## ¿Por qué usar GroupDocs.Redaction?
GroupDocs.Redaction ofrece una API unificada para la redacción de documentos en múltiples plataformas. Aquí hay algunas razones convincentes para elegir nuestra solución:

### Consistencia multiplataforma
Mantenga una lógica de redacción de documentos consistente en aplicaciones .NET y Java, reduciendo el tiempo de desarrollo y la carga de mantenimiento.

### Soporte extenso de formatos
Redacte información sensible de más de 30 formatos de documentos populares, incluyendo:

- Documentos PDF
- Formatos de Microsoft Office (Word, Excel, PowerPoint)
- Formatos OpenDocument
- Formatos de imagen (JPEG, PNG, TIFF)
- Formatos de correo electrónico (MSG, EML)
- Y muchos más

### Opciones integrales de redacción
- Redactar texto usando frases exactas, expresiones regulares o coincidencia sensible a mayúsculas/minúsculas  
- Limpiar propiedades de metadatos, comentarios e información oculta  
- Sanitizar o eliminar completamente imágenes con contenido sensible  
- Redactar anotaciones y comentarios que puedan contener información confidencial  
- Eliminar páginas completas o rangos de páginas que contengan material sensible  
- Aplicar políticas de redacción personalizadas para tipos de documento específicos  

### Enfoque en privacidad y seguridad
- Eliminación permanente de información sensible sin posibilidad de recuperación  
- Rasterización opcional para convertir documentos a PDFs basados en imágenes  
- Funciones de protección contra manipulaciones para evitar modificaciones no autorizadas  
- Sanitización completa del documento, incluidos metadatos y propiedades ocultas  

### Sin dependencias externas
GroupDocs.Redaction funciona sin requerir instalaciones de software externo como Microsoft Office, Adobe Acrobat u otras herramientas de terceros.

- **Agencias gubernamentales** que publican informes públicos mientras protegen secciones clasificadas.  
- **Empresas** que preparan documentos internos para socios externos sin exponer información propietaria.

## Preguntas frecuentes

**Q: ¿GroupDocs.Redaction admite archivos protegidos con contraseña?**  
A: Sí. Puede abrir PDFs, Word o archivos Excel encriptados proporcionando la contraseña durante el paso de carga.

**Q: ¿Puedo redactar contenido en lote en varios documentos?**  
A: Absolutamente. La API le permite iterar a través de una colección de archivos y aplicar la misma política de redacción a cada uno.

**Q: ¿La rasterización es opcional?**  
A: Sí. Puede elegir rasterizar todo el documento o solo páginas específicas cuando necesite una salida solo de imagen para mayor seguridad.

**Q: ¿Qué lenguajes de programación están cubiertos?**  
A: Los SDK centrales están disponibles para .NET (C# / VB.NET) y Java. También existen bibliotecas wrapper para otras plataformas.

**Q: ¿Cómo garantizo el cumplimiento de la redacción con GDPR?**  
A: Utilice la eliminación de metadatos incorporada y las funciones de eliminación permanente de contenido; la API garantiza que los datos redactados no puedan recuperarse, cumpliendo con el requisito de GDPR de “derecho al olvido”.
