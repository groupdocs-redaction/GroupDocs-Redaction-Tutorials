# 使用 GroupDocs.Redaction 實作安全文件遮蔽

探索針對 GroupDocs.Redaction 的完整知識庫，支援多平台，包括 .NET 與 Java。瀏覽涵蓋文字與中繼資料遮蔽、影像清理、註解移除以及進階遮蔽技術的各種教學。無論您是 .NET 或 Java 開發者，此資源中心皆提供 **實作安全文件遮蔽** 工作流程所需的工具與方法，保護敏感資訊同時維持文件完整性。

## 快速解答
- **什麼是安全文件遮蔽？** 一個永久移除或遮蔽文件中機密資料的過程，使其無法被恢復。  
- **支援哪些平台？** .NET 與 Java 皆支援，涵蓋超過 30 種檔案格式。  
- **是否需要外部軟體？** 不需要 — GroupDocs.Redaction 可在不安裝 Microsoft Office、Adobe Acrobat 或其他第三方工具的情況下運作。  
- **可以遮蔽 PDF 與影像嗎？** 可以，您可以遮蔽文字、中繼資料、影像、註解，甚至整頁。  
- **正式環境是否需要授權？** 正式使用需具備有效的 GroupDocs.Redaction 授權；亦提供免費試用版。

## 什麼是安全文件遮蔽？
安全文件遮蔽是一種永久刪除或隱蔽敏感資訊（如個人識別碼、財務資料或機密備註）的技術，從數位檔案中移除。與單純隱藏不同，遮蔽確保資料無法以任何方式恢復，符合 GDPR、HIPAA、PCI‑DSS 等合規標準。

## 為何使用 GroupDocs.Redaction？
GroupDocs.Redaction 為跨平台文件遮蔽提供統一的 API。以下是選擇我們解決方案的幾個重要理由：

### 跨平台一致性
在 .NET 與 Java 應用程式中維持一致的文件遮蔽邏輯，減少開發時間與維護成本。

### 廣泛格式支援
從 30 多種常見文件格式中遮蔽敏感資訊，包括：

- PDF 文件
- Microsoft Office 格式（Word、Excel、PowerPoint）
- OpenDocument 格式
- 影像格式（JPEG、PNG、TIFF）
- 電子郵件格式（MSG、EML）
- 以及其他多種格式

### 全面的遮蔽選項
- 使用精確字串、正規表達式或區分大小寫的匹配方式遮蔽文字  
- 清除中繼資料屬性、註解與隱藏資訊  
- 清理或徹底移除含有敏感內容的影像  
- 遮蔽可能包含機密資訊的註解與評論  
- 移除包含敏感內容的整頁或頁面範圍  
- 為特定文件類型套用自訂遮蔽政策  

### 隱私與安全為本
- 永久移除敏感資訊，無法復原  
- 可選的光柵化，將文件轉換為基於影像的 PDF  
- 防篡改功能，防止未授權的修改  
- 完整的文件清理，包含隱藏的中繼資料與屬性  

### 無外部相依性
GroupDocs.Redaction 可在不需安裝 Microsoft Office、Adobe Acrobat 或其他第三方工具的情況下運作。

## 常見問答

**Q: GroupDocs.Redaction 是否支援受密碼保護的檔案？**  
A: 是的。您可在載入階段提供密碼，以開啟加密的 PDF、Word 或 Excel 檔案。

**Q: 能否一次對多個文件批次遮蔽內容？**  
A: 當然可以。API 允許您遍歷檔案集合，對每個檔案套用相同的遮蔽政策。

**Q: 光柵化是可選的嗎？**  
A: 是的。當需要僅輸出影像以提升安全性時，您可以選擇將整份文件或特定頁面光柵化。

**Q: 支援哪些程式語言？**  
A: 核心 SDK 提供 .NET（C# / VB.NET）與 Java 版，亦有其他平台的封裝庫。

**Q: 如何確保遮蔽符合 GDPR 規範？**  
A: 使用內建的中繼資料移除與永久內容刪除功能；API 保證遮蔽資料無法復原，滿足 GDPR 「被遺忘權」的要求。
