# GroupDocs.Redaction for Java를 사용한 메타데이터 삭제 방법

이 가이드에서는 강력한 GroupDocs.Redaction 라이브러리를 사용하여 Java 문서에서 **메타데이터를 삭제하는 방법**을 알아봅니다. **문서 속성을 제거**하거나 **숨겨진 주석을 삭제**하거나 **Java 스타일로 문서를 보호**해야 할 경우, 이 튜토리얼은 숨겨진 정보를 식별하고 완전히 정리하는 단계까지 자세히 안내합니다. 개요를 마치면 메타데이터 삭제가 왜 중요한 보안 관행인지, 제공된 코드 샘플을 어떻게 자체 애플리케이션에 통합할 수 있는지 이해하게 됩니다.

## 메타데이터 삭제 – 빠른 개요

메타데이터는 종종 뒤에서 숨겨져 있습니다: 작성자 이름, 수정 이력, 사용자 정의 속성, 그리고 보이지 않는 주석까지. 이를 방치하면 민감한 비즈니스 데이터가 노출될 수 있습니다. GroupDocs.Redaction for Java는 다음과 같은 간단한 API를 제공합니다:

* **Extract document metadata** for inspection before removal. → **문서 메타데이터를 추출**하여 삭제 전에 검토합니다.  
* **Replace metadata text** with safe placeholders. → **메타데이터 텍스트를** 안전한 자리표시자로 **교체**합니다.  
* **Delete hidden comments** that may contain confidential notes. → **기밀 메모가 포함될 수 있는 숨겨진 주석을 삭제**합니다.  
* **Remove document properties** such as author, company, or custom tags. → **작성자, 회사, 사용자 정의 태그**와 같은 **문서 속성을 제거**합니다.  

이 기능들을 활용하면 배포, 보관 또는 규정 준수 감사를 앞두고 **문서를 안전하게 보호**할 수 있습니다.
